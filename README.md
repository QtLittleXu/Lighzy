# Lighzy

## 介绍

Lighzy 是一门简单易用的解释型编程语言，它是一种动态强类型定义语言，类似于 Python。Lighzy 即 "lighzy and easy"。

Lighzy 项目包含了 [Lighzy](https://github.com/QtLittleXu/lighzy-interpreter) 解释器（lighzy-interpreter）及计划开发的 Lighzy IDE（Lighdit）。
